# SparkJobs 🚀
> **من صناع عائلة SparkGen | Brought to you by the creators of SparkGen**

**SparkJobs** هو بوت توظيف ذكي ومجاني بالكامل يعمل على مدار الساعة (24/7) لمراقبة وظائفك المفضلة على أشهر المنصات العالمية، ومطابقتها مع سيرتك الذاتية (CV) باستخدام الذكاء الاصطناعي، وإرسال تنبيه فوري ومخصص لك على تليجرام.

---

## 🌍 المميزات الرئيسية / Key Features

* 🟢 **مجاني 100% للأبد (100% Free Forever):** يعمل بالكامل على خوادم GitHub Actions و Vercel المجانية بدون أي تكاليف استضافة أو اشتراكات.
* 🤖 **مطابقة ذكية بالـ AI (AI-Powered Matching):** يقوم Gemini 2.5 Flash بتحليل مدى توافق الوظيفة مع خبراتك وإعطائك نسبة توافق دقيقة.
* ✉️ **رسائل تقديم مخصصة (Custom Outreach Messages):** يولد لك رسالة تواصل احترافية جاهزة للإرسال لمسؤول التوظيف (Recruiter) لزيادة فرص قبولك.
* 🛡️ **كشف الوظائف الوهمية (Fraud & Scam Detection):** يقوم بتصفية الإعلانات القديمة والمكررة والشركات المشبوهة تلقائياً لحماية بياناتك.
* 🔒 **خصوصية تامة (Privacy First):** يتم تشفير مفاتيحك السرية كـ GitHub Secrets، ويتم مسح بياناتك الشخصية الحساسة (الاسم، الهاتف، الإيميل) من الـ CV قبل إرساله للـ AI.
* 📱 **تحكم كامل من تليجرام (Telegram Command Center):** إدارة ملفاتك الوظيفية وتشغيلها أو إيقافها بأوامر بسيطة من شات تليجرام.

## 🧙 دليل الإعداد السريع (في 5 دقائق) / Quick Setup Guide

لتشغيل نظام التوظيف الخاص بك مجاناً، يرجى اتباع الخطوات التالية:

### Step 1: إعداد تليجرام / Telegram Setup
1. افتح تطبيق تليجرام وابحث عن البوت الرسمي [@BotFather](https://t.me/botfather).
2. اكتب الأمر `/newbot` واتبع الخطوات لإنشاء بوت خاص بك، واحفظ الـ **API Token**.
3. ابحث عن بوت [@userinfobot](https://t.me/userinfobot) للحصول على الـ **Chat ID** الخاص بك.

### Step 2: إدخال مفتاح الذكاء الاصطناعي / Gemini API Key
1. اذهب إلى [Google AI Studio](https://aistudio.google.com/) وسجل دخولك.
2. اضغط على **"Get API Key"** وقم بإنشاء مفتاح مجاني جديد، ثم احفظه.

### Step 3: تفعيل التشغيل بضغطة زر / Deploy
1. افتح موقعنا الرسمي: `sparkjobs.app` (أو شغل الواجهة محلياً).
2. اتبع الـ Setup Wizard خطوة بخطوة وأدخل تفضيلات وظيفتك ومفاتيح الـ Tokens.
3. اضغط على **"بدء التفعيل والتشغيل 🚀"** وسيقوم الموقع بكل الخطوات الصعبة أوتوماتيكياً!
